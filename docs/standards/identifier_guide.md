# Identifier Guide

Use these identifiers in the universal secret key format:
`CLIENT_<CLIENTKEY>__<DOMAIN>__<KIND>__<IDENTIFIER>`

## Phones
- `PRIMARY_PHONE`
- `SECONDARY_PHONE`
- `OPS_CONTACT`

## Emails
- `PRIMARY_EMAIL`
- `BILLING_EMAIL`
- `OPS_EMAIL`

## Usernames / Handles
- `PRIMARY_USERNAME`
- `PRIMARY_HANDLE`
- `DISCORD_HANDLE`

## Logins
- `ADMIN_LOGIN`
- `OPS_LOGIN`

## Tokens / Keys
- `API_TOKEN`
- `WEBHOOK_SECRET`
- `CLIENT_SECRET`
